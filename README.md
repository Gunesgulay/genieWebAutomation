# Genie-Web-Automation

This project has been developed using Java and Selenium to perform automation tests on the Genie AI website. The tests have been written using Maven, Selenium WebDriver, and IntelliJ IDEA.

**Language:** Java (JDK 21)
**Library:** Selenium WebDriver
**Build Tool:** Maven
**IDE:** IntelliJ IDEA
**Design Pattern:** Page Object Model (POM)
